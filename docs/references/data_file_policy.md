# Data File Policy

작성일: 2026-03-13 (KST)

## 목적

- 데이터 파일의 추적/비추적 원칙과 운영 중 취급 규칙을 명확히 정의.

## Git 추적 정책

- 추적 제외(런타임 데이터):
  - `data/*`
  - `public/data/*` (실운영 데이터)
- 추적 유지(최소 부트스트랩/샘플):
  - `data/.gitkeep`
  - `public/data/mockup/**`

## 데이터 계층

1. 운영 런타임 데이터
   - 경로: `data/{season}/...`
   - 생성 주체: 크롤러/OpenAPI 배치
2. 프론트 샘플/테스트 데이터
   - 경로: `public/data/mockup/...`
   - 용도: UI 검증/문서 예시

## 배포 시 주의

1. 코드 배포와 운영 데이터 동기화 작업을 분리
2. 장애 대응 전에는 `data/` 백업 우선
3. `data/` 통째 덮어쓰기는 명시적 목적(마이그레이션/복구)일 때만 수행
