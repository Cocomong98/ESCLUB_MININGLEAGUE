# Reports Folder Guide

운영 노트/중간 보고서는 이 경로 하위에 날짜 기준으로 저장합니다.

## 권장 구조

`docs/reports/YYYY/MM/YYYY-MM-DD_<topic>.md`

예시:

- `docs/reports/2026/03/2026-03-13_csp_rollout_plan.md`
- `docs/reports/2026/03/2026-03-13_scheduler_tuning.md`

## 작성 규칙

1. 템플릿은 `docs/templates/work_note_template.md`를 사용
2. 작업 단위별로 파일 분리
3. 임시 메모라도 루트 디렉토리에 두지 않음
