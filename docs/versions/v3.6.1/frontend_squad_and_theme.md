# Frontend: Squad/UI Theme

## 배경

- 스쿼드 페이지 상단 레이아웃 정렬, 카드 비율, 다크모드 가독성 이슈가 반복 발생.

## 적용 범위

- `src/layouts/dashboard-squad/*`
- `src/components/pitch/*`
- 다크모드 관련 공통 스타일/카드 텍스트 보정

## 주요 변경

- 스쿼드 상단 좌/우 섹션 정렬 및 피치-카드 비율 조정
- 선수 카드 사이즈 반응형 축소 규칙 적용
- 선수 클릭 상세를 모달 기반으로 정리
- 다크모드에서 카드/모달 텍스트 대비 보정

## 검증 포인트

- PC/모바일에서 피치/선수 카드 비율 유지
- 다크모드에서 텍스트 미가시(흰 배경+흰 글자) 미발생
- 스쿼드 상세 모달 텍스트 대비 정상

## 롤백 포인트

- 스쿼드 상단 레이아웃 변경분
- PlayerChip 크기 토큰/스케일 규칙 변경분
